# Public deployment repository

This repository backs live Exec Ops Brief assets through GitHub Pages and
jsDelivr. Anything committed here, including deleted files in Git history, must
be treated as public and downloadable.

## Allowed

- Browser-delivered HTML, JavaScript, JSON, images, and video
- Public tools, job-board data, and published editorial assets
- Automation required to produce or refresh those public assets

## Keep elsewhere

- Paid, customer-only, or email-gated downloads
- Credentials, API keys, private feed URLs, and local configuration
- Source workbooks, private research, drafts, and internal operating records

Do not commit files under downloads/. Published downloads belong in the
configured Squarespace fulfillment system, and private source copies belong in
the business source folders outside this deployment repository.
